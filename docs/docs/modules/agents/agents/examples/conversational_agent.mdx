# Conversational Agent

이 예시는 채팅 모델을 위한 대화형 에이전트를 만드는 방법을 다룹니다. 채팅 특화 프롬프트를 사용할 것입니다.

<CodeBlock language="typescript">{Example}</CodeBlock>

````
에이전트 불러옴.
새로운 agent_executor 체인에 진입중...
{
    "action": "마지막 대답",
    "action_input": "안녕 Bob! 오늘은 어떻게 도와드릴까요?"
}
체인 종료.
출력 결과: 안녕 Bob! 오늘은 어떻게 도와드릴까요?
새로운 agent_executor 체인에 진입중...
{
    "action": "마지막 대답",
    "action_input": "너의 이름은 Bob 이야."
}
체인 종료.
출력결과: 너의 이름은 Bob 이야.
새로운 agent_executor 체인에 진입중...

```json
{
    "action": "검색",
    "action_input": "pomfret 의 날씨"
}
```

장마가 일찍찾아왔고...그리고 몇몇 소나기가 남아있는 흐린 날씨. 48F의 높은 온도. 풍속은 10~15mph. 비 올 확률 80% 입니다.

```json
{
    "action": "마지막 대답",
    "action_input": "장마가 일찍찾아왔고...그리고 몇몇 소나기가 남아있는 흐린 날씨. 48F의 높은 온도. 풍속은 10~15mph. 비 올 확률 80% 입니다."
}
```
체인 종료.
출력결과: 장마가 일찍찾아왔고...그리고 몇몇 소나기가 남아있는 흐린 날씨. 48F의 높은 온도. 풍속은 10~15mph. 비 올 확률 80% 입니다.
````

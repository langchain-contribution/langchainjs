# Unstructured

이 예시는 [Unstructured](../../../../../ecosystem/unstructured)을 사용하여 다양한 유형의 파일을 로드하는 방법에 대해 살펴봅니다. Unstructured은 현재 텍스트 파일, 파워포인트, html, pdf, 이미지 등등을 지원합니다.

## 준비

당신은 Docker를 사용하여 Unstructured를 동작시킬 수 있습니다. 우선, Docker를 설치해야 합니다. [여기](https://docs.docker.com/get-docker/)에서 Docker를 설치하는 방법을 찾을 수 있습니다.

```bash
docker run -p 8000:8000 -d --rm --name unstructured-api quay.io/unstructured-io/unstructured-api:latest --port 8000 --host 0.0.0.0
```

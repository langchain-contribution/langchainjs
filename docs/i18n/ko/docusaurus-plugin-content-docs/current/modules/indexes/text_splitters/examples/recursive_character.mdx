# `RecursiveCharacterTextSplitter`

권장되는 TextSplitter는 `RecursiveCharacterTextSplitter`입니다. 이것은 `"\n\n"`로 시작하여 `"\n"`, `" "` 순으로 다른 문자로 문서를 재귀적으로 분할합니다. 이 방법은 의미적으로 관련된 모든 콘텐츠를 같은 위치에 가능한 오래 유지하려고 하기에 좋습니다.

여기서 알아야 할 중요한 매개변수는 `chunkSize`와 `chunkOverlap`입니다. 'chunkSize'는 최종 문서의 최대 크기(문자 수 기준)를 제어합니다. `chunkOverlap`은 청크 사이에 얼마나 겹쳐야 하는지를 지정합니다. 이 설정은 텍스트가 이상하게 분할되지 않도록 하는 데 유용합니다. 아래 예시에서는 이 값을 작게 설정했지만(설명용으로) 실제로는 각각 `4000` 및 `200`으로 기본 설정되어 있습니다.

```typescript
import { RecursiveCharacterTextSplitter } from "langchain/text_splitter";

const text = `Hi.\n\nI'm Harrison.\n\nHow? Are? You?\nOkay then f f f f.
This is a weird text to write, but gotta test the splittingggg some how.\n\n
Bye!\n\n-H.`;
const splitter = new RecursiveCharacterTextSplitter({
  chunkSize: 10,
  chunkOverlap: 1,
});

const output = await splitter.createDocuments([text]);
```

위의 예시에서는 원문 텍스트 문자열을 분할하여 document 배열을 반환하고 있음을 알 수 있습니다. document를 직접 분할할 수도 있습니다.

```typescript
import { Document } from "langchain/document";
import { RecursiveCharacterTextSplitter } from "langchain/text_splitter";

const text = `Hi.\n\nI'm Harrison.\n\nHow? Are? You?\nOkay then f f f f.
This is a weird text to write, but gotta test the splittingggg some how.\n\n
Bye!\n\n-H.`;
const splitter = new RecursiveCharacterTextSplitter({
  chunkSize: 10,
  chunkOverlap: 1,
});

const docOutput = await splitter.splitDocuments([
  new Document({ pageContent: text }),
]);
```

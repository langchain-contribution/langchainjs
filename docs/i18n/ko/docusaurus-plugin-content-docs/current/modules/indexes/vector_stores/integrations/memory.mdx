# `MemoryVectorStore`

MemoryVectorStore는 임베딩을 메모리에 저장하고 가장 유사한 임베딩을 정확하게 선형 검색하는 인메모리 임시 벡터 저장소입니다. 기본 유사성 지표는 코사인 유사성이지만, [ml-distance](https://mljs.github.io/distance/modules/similarity.html)에서 지원하는 유사성 지표 중 하나로 변경할 수 있습니다.

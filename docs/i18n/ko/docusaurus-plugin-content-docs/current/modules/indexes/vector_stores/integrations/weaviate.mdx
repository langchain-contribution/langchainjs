# Weaviate

Weaviate는 객체와 벡터를 모두 저장하는 오픈 소스 벡터 데이터베이스로, 벡터 검색과 정형화된 필터링 기능을 제공합니다. LangChain은 Weaviate의 공식 Typescript 클라이언트인 `weaviate-ts-client` 패키지를 통해 Weaviate에 연결합니다.

LangChain은 벡터를 Weaviate에 직접 삽입하고, 주어진 벡터의 가장 가까운 이웃을 Weaviate에 쿼리하여 모든 LangChain 임베딩 통합 기능을 Weaviate와 함께 사용할 수 있습니다.

## 준비

```bash npm2yarn
npm install weaviate-ts-client graphql
```

로컬 또는 서버에서 Weaviate를 실행해야 하며, 자세한 내용은 [Weaviate 문서](https://weaviate.io/developers/weaviate/installation)를 참고하세요.

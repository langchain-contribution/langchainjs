# HNSWLib

:::tip Compatibility
Node.js에서만 사용 가능합니다.
:::

HNSWLib는 파일에 저장할 수 있는 in-memory vectorstore 입니다. 해당 문서에서는 [HNSWLib](https://github.com/nmslib/hnswlib)를 사용합니다.

## 설치

다음 명령어로 설치할 수 있습니다.

```bash npm2yarn
npm install hnswlib-node
```

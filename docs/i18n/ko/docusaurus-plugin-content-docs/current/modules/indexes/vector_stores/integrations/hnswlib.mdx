# HNSWLib

:::호환성 팁
Node.js 환경에서만 동작합니다.
:::

HNSWLib는 파일에 저장할 수 있는 인메모리 벡터 저장소입니다. [HNSWLib](https://github.com/nmslib/hnswlib)를 사용합니다.

## 준비

다음을 사용하여 설치할 수 있습니다.

```bash npm2yarn
npm install hnswlib-node
```

# Chroma

Chroma 는 오픈소스 Apache 2.0 임베딩 데이터베이스 입니다.

## 설치

1. 로컬 컴퓨터에서 Docker 로 Chroma 실행[docs](https://docs.trychroma.com/api-reference)
2. Chroma JS SDK를 설치

```bash npm2yarn
npm install -S chromadb
```

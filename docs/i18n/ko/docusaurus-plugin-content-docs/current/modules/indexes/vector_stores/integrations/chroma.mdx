# Chroma

Chroma는 오픈소스 Apache 2.0 임베딩 데이터베이스입니다.

## 준비

1. 컴퓨터에서 Docker로 Chroma 실행. [문서](https://docs.trychroma.com/api-reference)
2. Chroma JS SDK를 설치.

```bash npm2yarn
npm install -S chromadb
```

# 프롬프트 템플릿 (Prompt Templates)

:::info
[Conceptual Guide](https://docs.langchain.com/docs/components/prompts/prompt-template)
:::

프롬프트 템플릿 (Prompt Templates) 을 사용하면 템플릿을 사용하여 프롬프트를 생성할 수 있습니다. 이는 동일한 프롬프트를 여러 곳에서 사용하고 싶지만 특정 값이 변경될 경우에 유용합니다.
`Prompt Templates`는 LLM과 채팅 모델 모두 지원됩니다.

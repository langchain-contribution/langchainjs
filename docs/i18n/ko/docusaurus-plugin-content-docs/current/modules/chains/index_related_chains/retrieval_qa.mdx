# `RetrievalQAChain`

`RetrievalQAChain`은 위에서 설명한 `Retriever`와 QA 체인을 결합한 체인입니다.

`Retriever`에서 문서를 검색한 다음 `QA` 체인을 사용하여 검색된 문서를 기반으로 질문에 답변하는 데 사용됩니다.
